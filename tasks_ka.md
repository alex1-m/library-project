# Django Admin დავალება

თქვენი მიზანია დააკონფიგურიროთ Django-ს ადმინ პანელი არსებული მოდელებისთვის (`Author`, `Book`, `Review`). გამოიყენეთ ლექციაზე განხილული მასალა (ModelAdmin).

## დავალებები:

1.  ** რეგისტრაცია**:
    - დაარეგისტრირეთ სამივე მოდელი ადმინ პანელში.

2.  **AuthorAdmin-ის მორგება**:
    - `list_display`-ში გამოაჩინეთ ავტორის სახელი და რეიტინგი.
    - `search_fields` დააყენეთ სახელის მიხედვით.
    - `ordering` დააყენეთ რეიტინგის მიხედვით (კლებადობით).

3.  **BookAdmin-ის მორგება**:
    - `list_display`-ში გამოაჩინეთ სათაური, ავტორი, ჟანრი და ფასი.
    - დაამატეთ `list_filter` ჟანრისა და გამოქვეყნების თარიღის მიხედვით.
    - გამოიყენეთ `autocomplete_fields` ავტორის ასარჩევად (ამისთვის AuthorAdmin-ში უნდა გქონდეთ `search_fields`).

4.  **Inline რედაქტირება**:
    - შექმენით `ReviewInline` (`TabularInline`-ს გამოყენებით).
    - `BookAdmin`-ში დაამატეთ ეს inline-ი, რათა წიგნის გვერდიდანვე შეძლოთ მისი რევიუების ნახვა და დამატება.

5.  **Custom ველი (Optional)**:
    - `AuthorAdmin`-ში დაამატეთ მეთოდი `get_books_count`, რომელიც დაითვლის ავტორის წიგნების რაოდენობას და გამოაჩინეთ იგი `list_display`-ში.
    - N+1 პრობლემის თავიდან ასაცილებლად გამოიყენეთ `get_queryset` და `prefetch_related`.
